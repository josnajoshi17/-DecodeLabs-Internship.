# Project 1 - Rule Based AI Chatbot

## Description

This project was developed as part of the DecodeLabs Artificial Intelligence Internship.

The chatbot is a simple rule-based AI system that responds to predefined user inputs using if-else conditions.

## Features

* Handles greetings
* Responds to basic questions
* Accepts exit commands
* Runs in a continuous loop
* Uses decision-making logic

## Technologies Used

* Python

## Learning Outcomes

* Understanding control flow
* Working with loops
* Using conditional statements
* Building a basic AI chatbot
