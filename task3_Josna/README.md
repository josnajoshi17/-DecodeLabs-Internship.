# AI Recommendation Logic

## Project Overview

This project is a simple AI Recommendation System developed using Python. The system recommends movies to users based on their favorite genre.

## Features

* Takes user input for movie genre
* Matches user preferences with available genres
* Recommends relevant movies
* Uses simple recommendation logic and pattern matching

## Technologies Used

* Python

## How It Works

1. The user enters a favorite movie genre.
2. The program checks the genre in its database.
3. If the genre exists, it displays recommended movies.
4. If the genre is not available, an error message is shown.

## Sample Genres

* Action
* Comedy
* Horror
* Romance
