# AI Project 2 - Iris Flower Classification

## Objective
Classify Iris flowers using Machine Learning.

## Tools Used
- Python
- Scikit-learn
- Pandas
- NumPy

## Algorithm
K-Nearest Neighbors (KNN)

## Steps
1. Load Iris Dataset
2. Split Dataset
3. Apply StandardScaler
4. Train KNN Model
5. Predict Results
6. Evaluate using Accuracy, F1 Score, and Confusion Matrix

## Results
- Accuracy: 100%
- F1 Score: 1.0
